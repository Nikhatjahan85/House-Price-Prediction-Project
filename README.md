A Machine Learning based real estate analytics system for predicting property prices using housing features and intelligent predictive algorithms.




## Project Overview

The House Price Prediction Project is a Machine Learning application developed to estimate residential property prices based on different housing attributes.

The project includes:

• Dataset Generation  
• Data Preprocessing  
• Model Training  
• Price Prediction  
• Data Visualization  
• Interactive Streamlit Dashboard  

The model predicts house prices using features such as:

• Area Size  
• Bedrooms  
• Bathrooms  
• Parking  
• Furnishing Status  
• Property Age  
• Property Location  

The project also provides graphical analysis including heatmaps, feature importance graphs, prediction graphs, and market trend visualizations.

## Installation Guide

### 1. Clone Repository


git clone  https://github.com/Nikhatjahan85/House-Price-Prediction-Project




### 2. Open Project Folder


cd House-Price-Prediction-Project




### 3. Create Virtual Environment


python -m venv venv



### 4. Activate Virtual Environment

#### Windows


venv\Scripts\activate



### 5. Install Dependencies


pip install -r requirements.txt




## Running The Project

### Run Main Project


python main.py




### Run Streamlit Dashboard


streamlit run dashboard/app.py
